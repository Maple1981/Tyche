# Brennan Example Comparison Audit

Date: 2026-06-11

This audit compares public historical examples that appear in the supplied primary source with the charts currently generated by Tyche. It is a practical consistency check, not a new doctrine layer.

## Scope And Method

- Source layer: the supplied primary PDF, especially the example chapters and the appendix of charts and birth data.
- Tyche layer: current `app.js`, evaluated through the regression `window.TycheTest` API in browser mode so the results come from the same calculation path as the app.
- Tyche settings: tropical zodiac, Whole Sign Houses, traditional rulers, seven visible planets, local Astronomy Engine ephemerides, strict geometric sect by solar altitude.
- Included figures: public figures that both appear in the source examples and are already present in `HISTORICAL_PEOPLE`.
- Excluded material: anonymous/private examples, ancient example charts not in the historical-person archive, and names that appear only incidentally rather than as chart examples.

## Executive Finding

The overlap is broadly consistent. Tyche reproduces the source's stated rising signs, sect classifications, whole-sign placements, principal lot signs, and ruler-house logic for the checked examples.

The main substantive difference is George Lucas. The source explicitly treats his pre-sunrise chart as functionally diurnal for the lots and uses a slightly adjusted time. Tyche currently uses the recorded time and strict geometric sect, so it calculates the chart as nocturnal and places Fortune/Spirit differently. This is not an ephemeris error; it is a different technical convention.

Barack Obama is another edge case, but with a softer difference: the source treats the chart as nocturnal while noting reservations because the birth is just after sunset. Tyche also calculates it as nocturnal, but does not flag it as a widened sect-sensitive chart under the current thresholds.

## Technical Differences To Preserve

### George Lucas

The source uses an interpretive override:

- recorded time is treated as slightly rounded;
- time is moved forward by about 1.5 minutes;
- the chart is treated as effectively diurnal for the lots because the Sun is close to rising.

Tyche currently follows strict geometric sect. Therefore:

- Tyche correctly marks the recorded 5:40 chart as nocturnal;
- Tyche calculates night-formula Fortune and Spirit;
- the source's Spirit/Fortune signs for this example will not match Tyche unless Tyche gains an explicit, auditable sect/lot override for liminal charts.

Do not silently change Tyche's base rule to fit this example. If implemented later, it should be an advanced, clearly labeled override with the strict calculation still visible.

### Barack Obama

The source treats the chart as nocturnal but states that sect is debatable because the birth is just after sunset. Tyche also computes a night chart, but the solar altitude is outside the current liminal/sensitive thresholds. If desired, a future per-person historical note could surface this source-specific caution without changing the strict calculation.

## Practical Conclusion

For the overlapping figures, Tyche is using the same core natal criteria as the source: traditional rulers, whole-sign houses, sect, lots, triplicity rulers, ruler-of-place logic, bonification/maltreatment anchors, and the natal foundations for profections and zodiacal releasing.

The mismatch in George Lucas is useful: it confirms that Tyche is currently stricter than the source in edge cases where the author applies judgment, rectification, or a behavioral sect override. That should be treated as a documented interpretive divergence, not as a calculation bug.
