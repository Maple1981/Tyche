# Tyche

Tyche is a static Hellenistic astrology chart creator built with plain HTML, CSS, and JavaScript.

It calculates a browser-side natal chart with:

- Tropical zodiac and Whole Sign Houses by default.
- The seven visible planets as the strict base set.
- Ascendant, MC, IC, sect, dignities, lots, and configurations.
- English/Spanish UI.
- Day/night theme.
- Local serif typography and responsive mobile-first layout.

The astronomy engine is approximate and intended for study, exploration, and interface use. Critical charts should be checked against professional ephemerides.
